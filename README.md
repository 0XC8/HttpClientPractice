# HttpClientPractice
HttpClient采集项目最佳实践，该项目采用`.Net Core`开发，记录采集类项目经验的最佳实践。

## 包含的内容
 - HttpClientHelper 工具类封装，根据业务场景持续改进。
 - http请求性能测试
 - 采集时各种网络请求场景使用示例
 - 其它采集最佳实践与采坑经验
